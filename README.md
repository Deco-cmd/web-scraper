# Web Scraper Luitex 🛒


Automação com Python e Selenium para extrair nome e preço dos produtos da loja Luitex.

## 📦 Requisitos

- Python 3.x
- `selenium`
- `gspread`
- `oauth2client`

Instale tudo com:

```bash
pip install -r requirements.txt
